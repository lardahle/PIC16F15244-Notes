# PIC16F15244 Notes

Ongoing notes from learning embedded dev on the PIC16F15244 Curiosity Nano. Not a tutorial series — a record of what actually broke, why, and how it got fixed. Mostly MPLAB X + MCC Melody, sometimes bare-metal when MCC's abstraction got in the way of understanding what was actually happening on the silicon.

If you landed here from a search engine because your serial terminal is doing something inexplicable, check the [gotchas index](#gotchas) below first.

## Hardware

- [PIC16F15244 Curiosity Nano](https://www.microchip.com/en-us/development-tool/ev09z19a) (DM... EV09Z19A)
- MPLAB X IDE 6.15+
- XC8 Compiler 2.45+
- MPLAB Code Configurator (MCC) — Melody

## Gotchas

The expensive lessons, each with a standalone writeup so they're individually linkable/searchable.

| Symptom | Cause | Doc |
|---|---|---|
| Serial terminal prints one repeating character, each on its own line | `EUSART1_Write()` generated by MCC Melody is non-blocking — it doesn't wait for the transmit shift register to clear | [docs/gotchas/eusart1-write-non-blocking.md](docs/gotchas/eusart1-write-non-blocking.md) |

More to come as I break more things.
